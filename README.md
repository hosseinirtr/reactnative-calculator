<h1><strong>Installation</strong></h1>

If you Want to Test app on You phone :

Step 1:
Install Node js 

Step2:
run this code on your Terminal or CMD:

$ adb devices 
$ adb -s <DeviceName> reverse tcp:8081 tcp:8081    
$ npm install
$ npm react-native run-android

(on linux)
$ npm start
